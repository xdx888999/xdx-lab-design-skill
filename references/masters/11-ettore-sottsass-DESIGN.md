# Design System: Ettore Sottsass Method
# 埃托雷·索特萨斯设计方法

> 风格定位：反中性、强色彩、图案化结构和物件人格。让设计从“正确”中逃出来，变成有情绪的对象。
> 使用边界：参考其后现代和 Memphis 语境中的方法论，不复制具体家具、纹样或商业产品。

## 1. Visual Theme & Philosophy

- 核心气质：挑衅、玩世不恭、彩色、后现代、对象感。
- 设计目标：打破冷静现代主义，用颜色和图案让对象有性格。
- 适用场景：潮流品牌、展览、音乐活动、创意社群、艺术电商、实验海报。

## 2. Color Palette & Roles

```text
Background:   #F8E9D2
Text primary: #111111
Text muted:   #5B5148
Pink:         #EF476F
Teal:         #00A6A6
Yellow:       #FFD166
Purple:       #5A189A
Pattern line: #111111
```

## 3. Typography Rules

- Heading: `Archivo Black`, `Space Grotesk`, `Arial Black`, sans-serif; weight 800-900.
- Body: `Space Grotesk`, `Inter`, sans-serif; weight 400-500.
- 标题可大、怪、块状，但要控制字数。
- 不要使用传统奢华衬线字体。

## 4. Layout & Component Rules

- 使用斜线、棋盘格、波浪、圆柱感、错位台座和厚边框。
- 卡片像小型展台或物件底座，允许鲜明背景和几何分层。
- 布局可以不对称，但每个色块要有明确边界。
- CTA 可以像贴纸或按钮块，边框粗、颜色强。

## 5. Imagery & Iconography

- 插图优先于真实摄影；使用图案、物件、色块、立体感几何。
- 图片可做硬裁切和厚描边，避免自然主义。
- 图标像玩具或展览符号，带一点怪异人格。

## 6. Do's and Don'ts

Do:
- 让颜色和图案成为结构。
- 用幽默和反差打破中性。
- 保持信息层级，不要让热闹盖住内容。

Don't:
- 不要做成普通彩色卡片。
- 不要滥用随机图案。
- 不要复制 Memphis 具体纹样。

## 7. Output Behavior

- 海报：适合强标题、图案背景和对象式主视觉。
- 网站：适合创意品牌首页，但正文区要降低图案密度。
- PPT：适合章节页、案例页和活动宣传，不适合严肃财务报告。

## 8. Agent Prompt Guide

```text
使用 Ettore Sottsass 方法设计。用 #F8E9D2 背景、#EF476F、#00A6A6、#FFD166、#5A189A 和粗黑线，创造后现代对象感。图案要参与结构，不要只是贴背景；整体要大胆但仍可读。
```
