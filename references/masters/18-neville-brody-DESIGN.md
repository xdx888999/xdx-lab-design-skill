# Design System: Neville Brody Method
# 内维尔·布罗迪设计方法

> 风格定位：激进字体、杂志系统和数字时代的字形实验。让字体表现文化身份，而不是只承载文本。
> 使用边界：参考其字体和编辑方向方法，不复制 The Face、Arena、FUSE 或具体字体作品。

## 1. Visual Theme & Philosophy

- 核心气质：后朋克、编辑性、锋利、实验、类型系统。
- 设计目标：让字体成为文化态度和版面结构。
- 适用场景：时尚杂志、音乐、文化评论、字体实验、品牌视觉系统。

## 3. Typography Rules

- Heading: `Space Grotesk`, `Unbounded`, `Eurostile`, sans-serif; weight 700-900.
- Body: `Inter`, `Space Grotesk`, sans-serif; weight 400.
- 标题可被压缩、扩展、分段、旋转，但必须建立系统。
- 字体实验要有重复规则，不能每页都随机。

## 4. Layout & Component Rules

- 使用强网格和破网格并存：标题激进，信息块严谨。
- 黑底、白字、红/酸绿强调适合高对比文化页面。
- 卡片像杂志栏、唱片封套、字体样张。
- 适合加入编号、条码、纵向字、巨型字母。

## 5. Imagery & Iconography

- 图片可高对比、黑白、金属感或强裁切。
- 图标使用锐利几何、工业符号、数字标记。
- 视觉资产要服务编辑态度，不要普通装饰。

## 6. Do's and Don'ts

Do:
- 让字体有态度。
- 建立可重复的类型系统。
- 在实验和可读之间做明确取舍。

Don't:
- 不要把所有元素都做成同等激进。
- 不要用过度霓虹做成赛博朋克。
- 不要复制具体杂志风格。

## 7. Output Behavior

- 网站：适合文化媒体、时尚品牌、音乐专题。
- 海报：适合字体主导的强态度视觉。
- PPT：适合趋势、品牌、文化研究，不适合基础教学正文。

## 8. Agent Prompt Guide

```text
使用 Neville Brody 方法设计。黑底白字，加入 #FF3B30 和 #B9FF00，使用激进字体、巨型字母、纵横交错的编辑系统。字体要成为文化态度，但正文信息必须有稳定网格支撑。
```
