# Design System: Ikko Tanaka Method
# 田中一光设计方法

> 风格定位：日本传统图像、现代几何和高克制色彩。把能乐、琳派、文字和现代网格融合成新的文化视觉。
> 使用边界：参考其传统与现代融合的方法，不复制具体剧场海报、品牌识别或人物构图。

## 1. Visual Theme & Philosophy

- 核心气质：日本性、几何、优雅、舞台感、现代。
- 设计目标：用有限形状和强色彩表达传统文化的当代表情。
- 适用场景：日本文化品牌、文旅、剧场、展览、精品餐饮、传统工艺。

## 2. Color Palette & Roles

```text
Background:   #F7F2E8
Text primary: #111111
Text muted:   #6B6258
Vermilion:    #E60012
Indigo:       #005BAC
Gold:         #F5B400
Purple:       #6F2DA8
Paper line:   #D8CBB8
```

## 3. Typography Rules

- Japanese heading: `Noto Serif JP`, `Shippori Mincho`, serif; weight 600-700.
- Latin heading: `Inter`, `Helvetica Neue`, sans-serif; weight 600.
- Body: `Noto Sans JP`, `Inter`, sans-serif.
- 中日英混排要有清楚比例，日文标题可成为图形结构。

## 4. Layout & Component Rules

- 使用大几何块、垂直轴线、舞台式留白和少量传统色。
- 可把人物、面具、花、扇、圆形转译成几何符号。
- 卡片像展览标签，内容少、边距大。
- 网格要现代，但气质要有日本传统节制。

## 5. Imagery & Iconography

- 优先使用几何化文化符号，不直接堆砌浮世绘或和风素材。
- 图标保持扁平、干净、色块明确。
- 可以使用纸纹，但必须非常克制。

## 6. Do's and Don'ts

Do:
- 把传统符号抽象成几何。
- 控制颜色数量，让红/蓝/金产生舞台感。
- 保留大量留白。

Don't:
- 不要把日本风格做成旅游纪念品。
- 不要混用浮世绘、动漫、禅意等不相干符号。
- 不要复制具体能乐人物海报。

## 7. Output Behavior

- 海报：适合文化活动和展览。
- 网站：适合日本餐饮、工艺、博物馆专题。
- PPT：适合文化介绍，章节页可使用几何符号。

## 8. Agent Prompt Guide

```text
使用 Ikko Tanaka 方法设计。将日本传统元素抽象为现代几何，用 #F7F2E8、#E60012、#005BAC、#F5B400 和少量紫色构成舞台式版面。中日英排版要克制、优雅、留白充足，不要堆砌和风素材。
```
