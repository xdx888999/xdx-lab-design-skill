# Design System: Paula Scher Method
# 宝拉·谢尔设计方法

> 风格定位：字体即图像，品牌即城市声音。用巨大的文字、节奏和公共文化能量建立可识别系统。
> 使用边界：参考其文化机构、剧场和品牌系统方法，不复制 Public Theater 等具体识别。

## 1. Visual Theme & Philosophy

- 核心气质：大胆、城市、公共、活力、类型驱动。
- 设计目标：让字体从信息载体变成主视觉和空间结构。
- 适用场景：剧场、音乐节、文化活动、城市品牌、campaign、展览导视。

## 2. Color Palette & Roles

## 3. Typography Rules

- Heading: `Archivo Narrow`, `Oswald`, `Arial Narrow`, sans-serif; weight 700-900.
- Body: `Inter`, `Arial`, sans-serif; weight 400-500.
- 同一标题可混合字号、字重、宽度，但必须形成节奏。
- 字体可以占据画面 50% 以上。

## 4. Layout & Component Rules

- 使用大字、纵横交错、重复、街头海报感和强色块。
- 卡片可以像票根、街贴、节目单。
- 信息层级要像活动海报：谁、什么、何时、在哪里、行动。
- CTA 要像售票按钮或报名入口，醒目直接。

## 5. Imagery & Iconography

- 图片可被大字压住、裁切或作为文字背景。
- 图标少用；文字本身就是图像。
- 适合用场地照片、舞台、城市纹理，但要强裁切。

## 6. Do's and Don'ts

Do:
- 让排版承担主视觉。
- 用城市和公共文化的能量组织画面。
- 保持关键信息可读。

Don't:
- 不要把大字做成没有节奏的堆叠。
- 不要使用柔弱、过细的字体。
- 不要复制具体剧场视觉系统。

## 7. Output Behavior

- 海报：最适合，标题就是主视觉。
- 网站：适合活动页、文化机构首页、campaign landing。
- PPT：适合发布会、活动方案和品牌提案。

## 8. Agent Prompt Guide

```text
使用 Paula Scher 方法设计。让字体成为主视觉，用大字号、窄体粗字、重复和强色块制造城市文化能量。使用 #E31B23、#FFD100、#0057B8、#FF5A1F。关键信息必须可读，不要复制 Public Theater。
```
