# Design Master Index
# 20 位设计大师方法论索引

> 这份索引用于在用户指定“像某位设计大师那样设计”时快速路由。这里整理的是公开作品和设计思想中的方法论，不用于复制具体作品、商标或受保护的商业识别系统。

---

## 使用规则

- 如果用户指定设计大师姓名、作品方向或相近描述，先匹配本索引，再读取对应 `DESIGN.md`。
- 如果用户没有指定大师，只是要求“更经典 / 更大师 / 更有设计史感”，可先推荐 2-3 个方法论方向让用户选择。
- 对在世设计师或仍受商业保护的品牌系统，只能提取原则、构图逻辑、信息层级和视觉语法，不要声称“复刻本人风格”。
- 大师方法论可与 50 个精选风格叠加使用，例如“浮世绘 + Saul Bass 的电影海报式构图”，但必须保证内容可读、版权边界清楚。

## 推荐话术

当用户没有指定大师但希望“更有大师感”时，可使用：

```text
我建议从这几个设计大师方法论里选一个：

1. Dieter Rams：适合产品、工具和硬件，克制、清晰、功能优先。
2. Saul Bass：适合海报、封面和演示开场，用强符号和电影感构图快速抓人。
3. Paula Scher：适合文化活动和品牌 campaign，把字体做成主视觉，更有现场感。

你可以回复编号，也可以直接说你想参考哪位设计大师的方法。
```
