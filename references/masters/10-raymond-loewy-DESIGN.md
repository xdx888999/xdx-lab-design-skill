# Design System: Raymond Loewy Method
# 雷蒙德·洛维设计方法

> 风格定位：MAYA 原则、流线型工业美学和大众可接受的先进感。让产品看起来更快、更可靠、更想被使用。
> 使用边界：参考其工业设计原则和流线型语言，不复制具体交通工具、包装或品牌标志。

## 1. Visual Theme & Philosophy

- 核心气质：流线、现代、商业吸引力、速度、可信。
- 设计目标：在“足够先进”和“容易接受”之间取得平衡。
- 适用场景：交通、硬件、消费电子、包装、品牌升级、产品发布页。

## 3. Typography Rules

- Heading: `Avenir`, `Montserrat`, `Inter`, sans-serif; weight 600-800.
- Body: `Inter`, `Avenir`, sans-serif; weight 400.
- 数字和规格可使用 `Roboto Condensed` 或 `IBM Plex Sans Condensed`。
- 标题可略带横向延展感，但不要使用廉价复古字体。

## 4. Layout & Component Rules

- 使用横向动线、圆角长条、速度线、剖面参数卡。
- 产品图应成为中心，信息像工业标牌一样环绕。
- 卡片半径 6px，边框轻，阴影像产品摄影的自然投影。
- CTA 要直接，强调“试用 / 预约 / 查看规格”。

## 5. Imagery & Iconography

- 使用高质量产品图、金属材质、剖面图、流线轮廓。
- 图标使用简化工业符号，不使用复杂插画。
- 可把产品 silhouette 作为主视觉。

## 6. Do's and Don'ts

Do:
- 在先进感和熟悉感之间平衡。
- 强调外形、性能、可靠性和商业价值。
- 用流线引导视线。

Don't:
- 不要做成过度科幻。
- 不要使用随机复古徽章。
- 不要让装饰线条影响产品可读性。

## 7. Output Behavior

- 网站：适合产品发布页、交通工具展示、硬件官网。
- PPT：适合产品升级、设计提案、概念车/硬件说明。
- 海报：适合流线 silhouette + 规格标签。

## 8. Agent Prompt Guide
