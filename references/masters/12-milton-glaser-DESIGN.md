# Design System: Milton Glaser Method
# 米尔顿·格雷泽设计方法

> 风格定位：人文、插画、城市文化和可亲近的智慧。用图像讲故事，让商业传播也有温度。
> 使用边界：参考其海报、编辑和城市文化方法，不复制 I Love NY、Bob Dylan 等具体作品。

## 1. Visual Theme & Philosophy

- 核心气质：人文、乐观、机智、绘画性、文化感。
- 设计目标：用插画和排版让复杂文化信息变得可记忆、可喜欢。
- 适用场景：文化海报、城市品牌、编辑视觉、书店、餐饮、艺术节。

## 2. Color Palette & Roles

```text
Background:   #F5E6C8
Text primary: #111111
Text muted:   #665A4A
Red:          #E63946
Blue:         #1D3557
Teal:         #2A9D8F
Orange:       #F4A261
Cream:        #FFF6E3
```

## 3. Typography Rules

- Heading: `Fraunces`, `Cooper Black`, `Georgia`, serif; weight 700-900.
- Body: `Work Sans`, `Inter`, sans-serif; weight 400.
- 字体可有复古和插画气质，但必须清晰。
- 标题适合圆润、有表情的字形，不要过冷。

## 4. Layout & Component Rules

- 使用主插画 + 大标题 + 简洁说明的海报结构。
- 允许曲线、手绘线条、彩色块和温暖纸张纹理。
- 卡片可像编辑版块，带插图小角标和引用。
- 信息应有故事顺序：主题、情绪、细节、行动。

## 5. Imagery & Iconography

- 使用原创插画、手绘人物、城市符号、文化物件。
- 可用一笔线、拼贴和复古印刷质感。
- 图像要有观念，不只是装饰。

## 6. Do's and Don'ts

Do:
- 给视觉一个人文故事。
- 用温暖色彩和插画建立亲近感。
- 保持商业传播的清楚和记忆点。

Don't:
- 不要复制著名标志或海报。
- 不要做成普通复古风。
- 不要让插画复杂到影响标题。

## 7. Output Behavior

- 海报：最适合，用一个概念插画支撑主标题。
- 网站：适合文化机构、城市活动、编辑专题。
- PPT：适合开场、人物故事和文化案例。

## 8. Agent Prompt Guide

```text
使用 Milton Glaser 方法设计。用温暖纸色、手绘插画、人文故事和可亲近的标题字体组织画面。色彩使用 #E63946、#1D3557、#2A9D8F、#F4A261。不要复制 I Love NY 或任何具体作品，要原创视觉隐喻。
```
