# Design System: Saul Bass Method
# 索尔·巴斯设计方法

> 风格定位：用极简符号、剪纸形态和电影感节奏，在第一眼传达故事本质。
> 使用边界：参考其片头、海报和企业识别中的“视觉短句”方法，不复制具体电影海报或标题序列。

## 1. Visual Theme & Philosophy

- 核心气质：戏剧性、直接、粗粝、运动感、强符号。
- 设计目标：把复杂叙事压缩成一个能动起来的视觉短句。
- 适用场景：电影海报、活动主视觉、PPT 开场、品牌 campaign、视频封面。

## 2. Color Palette & Roles

```text
Background:   #F4E5CF  aged paper
Text primary: #0B0B0B
Text muted:   #5B5248
Red:          #D7261E
Mustard:      #F2B705
Green:        #2F5D50
Cream:        #FFF3D8
```

## 3. Typography Rules

- Heading: `Oswald`, `Futura Condensed`, `Arial Narrow`, sans-serif; weight 700.
- Body: `Libre Franklin`, `Inter`, sans-serif; weight 400-500.
- 标题允许轻微错位、剪纸感和不完全规整，但必须可读。
- 字体要像片名字卡一样有进入故事的力量。

## 4. Layout & Component Rules

- 使用大面积平涂色块、斜切形态、粗黑轮廓。
- 主视觉可以突破网格，但文字信息必须有清楚落点。
- 卡片少用；更适合海报式分区、横幅、标签和章节条。
- 用强对角线或中心符号制造运动感。

## 5. Imagery & Iconography

- 优先使用剪影、断裂形、手工纸边、粗线条符号。
- 人物和物体只保留最能讲故事的特征。
- 可把图形拆成可运动的层，方便做片头或演示转场。

## 6. Do's and Don'ts

Do:
- 用一个符号讲清主题。
- 提高对比和节奏，保持画面有戏剧张力。
- 让封面远看就能识别主题。

Don't:
- 不要用写实电影剧照堆叠。
- 不要把纹理做得脏乱影响文字。
- 不要模仿具体电影作品构图。

## 7. Output Behavior

- PPT：封面和章节页可用电影片头式构图，正文页保持可读。
- 网站：Hero 使用强符号和横向运动感，后续区块回到清晰信息。
- 海报：优先 2-3 色平涂，避免照片依赖。

## 8. Agent Prompt Guide

```text
使用 Saul Bass 方法设计。用老纸色 #F4E5CF、黑色剪影、#D7261E 和 #F2B705 构成强烈电影海报感。先创造一个原创视觉符号，再用粗体窄字和斜向构图组织信息。不要复制任何具体电影海报。
```
