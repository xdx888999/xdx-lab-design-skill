# Design System: David Carson Method
# 大卫·卡森设计方法

> 风格定位：解构编辑、冲浪/音乐文化、质感噪声和有意图的可读性破坏。规则可以被打破，但必须有内容理由。
> 使用边界：参考其编辑实验和文化语境，不复制 Ray Gun 具体版式或摄影处理。

## 1. Visual Theme & Philosophy

- 核心气质：反网格、粗粝、主观、音乐性、地下文化。
- 设计目标：让版面情绪和内容气质合一，而不是追求中性可读。
- 适用场景：音乐、青年文化、独立杂志、实验海报、艺术作品集、街头品牌。

## 2. Color Palette & Roles

```text
Background:   #EAE1D0
Text primary: #111111
Text muted:   #6B7280
Dust:         #A08B73
Rust:         #C2410C
Deep gray:    #394150
Paper:        #F7F2E7
```

## 3. Typography Rules

- Heading: `Archivo Condensed`, `Impact`, `Helvetica Neue`, sans-serif; weight 700-900.
- Body: `Inter`, `Georgia`, sans-serif/serif mix allowed when intentional.
- 可以使用错位、重叠、旋转、裁切，但核心信息必须保留阅读路径。
- 字体选择要来自内容情绪，不是随机混搭。

## 4. Layout & Component Rules

- 允许打破网格：漂移文本、极端裁切、图片噪声、层叠。
- 必须有一个稳定锚点，例如主标题、照片或 CTA。
- 卡片可以像杂志剪贴、扫描纸、贴片。
- 不适合严肃表格；适合专题页和封面。

## 5. Imagery & Iconography

- 使用粗颗粒摄影、扫描纹理、手工痕迹、错位裁切。
- 图片可以低保真，但不能低质量到无法辨认主题。
- 图标少用；可用手写标记、箭头、编号。

## 6. Do's and Don'ts

Do:
- 让每个破坏动作都有内容理由。
- 保留一条核心阅读路径。
- 使用质感增强情绪。

Don't:
- 不要把所有文字都做成不可读。
- 不要把解构当成随机混乱。
- 不要用于医疗、金融、法律等高信任场景。

## 7. Output Behavior

- 海报：适合音乐和街头文化。
- 网站：适合作品集、专题页；导航和 CTA 必须仍清楚。
- PPT：只建议封面、章节、案例页使用，正文页要收敛。

## 8. Agent Prompt Guide

```text
使用 David Carson 方法设计。采用解构编辑、强裁切、扫描质感、漂移文字和粗粝摄影，但必须保留核心阅读路径。色彩以纸色、黑、灰、锈红为主。每个“破坏规则”的动作都要服务内容情绪。
```
