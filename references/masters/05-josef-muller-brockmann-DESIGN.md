# Design System: Josef Mueller-Brockmann Method
# 约瑟夫·穆勒-布罗克曼设计方法

> 风格定位：瑞士网格、客观排版和音乐般的结构节奏。形式服从信息，秩序产生美感。
> 使用边界：参考其国际主义平面设计方法，不复制具体音乐会海报或书籍版式。

## 1. Visual Theme & Philosophy

- 核心气质：客观、精密、现代、理性、节奏化。
- 设计目标：让信息通过网格、比例、对齐和节奏自然显现。
- 适用场景：学术报告、展览海报、数据图、建筑网站、设计学院材料。

## 2. Color Palette & Roles

```text
Background:   #FFFFFF
Text primary: #111111
Text muted:   #666666
Grid line:    #D9D9D9
Accent red:   #E3001B
Accent blue:  #0057A6
Accent gray:  #F2F2F2
```

## 3. Typography Rules

- Heading: `Helvetica Neue`, `Arial`, `Inter`, sans-serif; weight 500-700.
- Body: `Helvetica Neue`, `Arial`, `Inter`, sans-serif; weight 400.
- 字号层级必须来自比例，而不是随意调大。
- 标题和正文优先左对齐，避免居中装饰。

## 4. Layout & Component Rules

- 必须使用明确网格：3/6/12 列都可，但所有元素要对齐。
- 使用圆形、斜线、重复条带表现节奏和结构。
- 图表和文本共享同一基线系统。
- 组件边框细、背景轻，不使用厚重阴影。

## 5. Imagery & Iconography

- 摄影可做高对比黑白裁切，作为结构元素。
- 图形使用几何抽象，不使用插画化装饰。
- 数据图优先使用线、点、圆和模块化标签。

## 6. Do's and Don'ts

Do:
- 用网格解决版面，不靠感觉摆放。
- 让负空间成为结构的一部分。
- 使用数学比例处理标题、图形和正文。

Don't:
- 不要使用随机旋转和破碎排版。
- 不要为了活泼加入无意义色块。
- 不要用花体、手写体或过度装饰字体。

## 7. Output Behavior

- 海报：大网格 + 少量几何 + 明确信息层级。
- PPT：适合数据、研究、概念推导；每页网格一致。
- 网站：适合机构、建筑、设计教育，不适合强情绪消费页。

## 8. Agent Prompt Guide

```text
使用 Josef Mueller-Brockmann 方法设计。白底、黑字、严格瑞士网格，少量 #E3001B 或 #0057A6 做结构强调。使用 Helvetica/Inter，所有标题、正文、图表必须对齐到同一网格。设计应客观、精密、有节奏。
```
