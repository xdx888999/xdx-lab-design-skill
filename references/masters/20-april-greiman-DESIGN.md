# Design System: April Greiman Method
# 艾普丽尔·格雷曼设计方法

> 风格定位：早期数字拼贴、空间层叠、身体与技术的混合。把电脑当成新媒介，而不是传统工具的替代品。
> 使用边界：参考其数字图形实验方法，不复制具体海报、影像或个人作品。

## 1. Visual Theme & Philosophy

- 核心气质：数字、层叠、身体感、空间、实验。
- 设计目标：展示技术如何改变视觉空间和阅读体验。
- 适用场景：数字海报、创意作品集、实验网页、科技文化展、艺术教育。

## 2. Color Palette & Roles

```text
Background:   #F3E8FF
Text primary: #111111
Text muted:   #5B5266
Magenta:      #FF2D95
Cyan:         #00AEEF
Yellow:       #FFD100
Purple:       #8E44AD
White:        #FFFFFF
```

## 3. Typography Rules

- Heading: `Syne`, `Space Grotesk`, `Helvetica Neue`, sans-serif; weight 700-900.
- Body: `Inter`, `Helvetica Neue`, sans-serif; weight 400.
- 可使用纵向、斜向和层叠文字，但要给读者明确入口。
- 字体可以与图像互相穿插。

## 4. Layout & Component Rules

- 使用 collage、扫描线、透明层、像素感、窗口层叠和非线性阅读。
- 画面可以像数字工作台，但必须控制层级。
- 卡片可像浮动窗口或信息碎片。
- 适合视差滚动、鼠标跟随和分层动效。

## 5. Imagery & Iconography

- 使用身体局部、扫描图、电脑界面片段、色彩通道错位。
- 图像可半透明、叠印、网点化、低分辨率化。
- 图标可以像早期数字界面符号，但要原创。

## 6. Do's and Don'ts

Do:
- 把数字媒介特性作为视觉语言。
- 用层叠展示信息关系。
- 在实验中保留主标题和关键路径。

Don't:
- 不要做成普通 Y2K 渐变。
- 不要堆太多滤镜导致主题消失。
- 不要把实验视觉用于高信任严肃界面。

## 7. Output Behavior

- 网站：适合 full-bleed 实验首页和作品集。
- 海报：适合数字艺术、展览、技术文化。
- PPT：适合开场和概念页，正文页要降低层叠密度。

## 8. Agent Prompt Guide

```text
使用 April Greiman 方法设计。采用数字拼贴、透明层、扫描线、窗口碎片和洋红/青/黄高饱和色，表现技术与身体、空间、图像的混合。实验可以大胆，但必须保留清楚阅读入口。
```
